+ run

bongSDK 开启扫描

+ (void)run

Discussion

bongII 的同步功能需要保持蓝牙开启并持续扫描， 当扫描到广播数据时，会回调 bongSDKDidScanbongIIWithPressYesKey:， 此时，可以根据返回的按键类型做处理或直接同步数据。

该方法运行结果可在委托方法 bongSDKDidRun: 获取。
